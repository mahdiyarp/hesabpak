hesabpak

این سایت آماده است تا برنامه حساب‌پاک را میزبانی کند. برای نصب خودکار روی سرور، از اسکریپت deploy_site.sh در ریشه‌ی مخزن استفاده کنید.

نصب خودکار

  1. یک ماشین لینوکسی (Ubuntu) آماده کنید.
  2. اسکریپت را با دستور sudo bash deploy_site.sh اجرا کنید.
  3. اسکریپت مخزن را از GitHub کلون می‌کند، محیط مجازی می‌سازد، وابستگی‌ها را نصب می‌کند و یک سرویس systemd برای اپ ایجاد می‌کند و Nginx را پیکربندی می‌کند.

نام دامنه: hesabpak.com

برای شخصی‌سازی یا سوالات بیشتر، فایل README_DEPLOY.md را ببینید.
